# Lab 5 Repository

This repository contains the website created in `lab_4` and published as part of Laboratory Work 5.

## Project

Interactive personal CV website with:

- responsive layout
- light and dark theme switcher
- scheduled theme indicator
- browser storage support
- comments and feedback interactions

## Files

- `index.html` - main page markup
- `styles.css` - visual styles and responsive layout
- `script.js` - interactive behavior

## Environments

- `main` -> production deployment
- `staging` -> preview deployment for intermediate verification on Vercel

## Run Locally

Open `index.html` in a browser.
